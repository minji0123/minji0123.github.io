## 시도한 방법

```bash
npm install
```

npm 설정에 문제가 생겼다고 판단해서  
다시 npm 을 설치했습니다.  

하지만 설치가 제대로 되지 않았습니다.

## 해결방법

그래서 저는 오류를 우회해서 설치하는 방법을 사용했습니다.

```bash
npm install --force
```

![](/src/images/../../src/images/2022/2022-05-29-17-36-20.png)

각종 경고는 떴지만 그래도 무사히 설치는 되었습니다...😦
